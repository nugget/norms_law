Norm's Law

This is a Chrome browser extension that will fix posts on the Motley Fool message board whenever Norm (rev2217) erroneously refers to "Natural Law" when, in fact, he means "NORM'S LAW."

Installation Instructions

  • Download the crx file
  • In Chrome, choose Window > Extensions
  • Drag the norms_law.crx file from your Downloads folder into the Chrome Extensions window
